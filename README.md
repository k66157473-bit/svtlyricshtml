# svtlyricshtml
## 説明
このレポジトリは、**SEVENTEENのパート割(歌割)を表示する際につかえるコードを設置したものです**
コードから歌詞や色わけを入れ替えるだけで簡単に使えます
## 使い方
*1.まず、レポジトリからindex.htmlと、menber.zipをダウンロードしてください
 *   2.次に、そのファイルらをサーバーにアップロードしてください
  *  3.index.htmlの中の歌詞の部分と 歌詞についている class名(例:scoups)などを該当するメンバーに変えてください
   * 4.完成です
