# svtlyricshtml
## 説明
このレポジトリは、**SEVENTEENのパート割(歌割)を表示する際につかえるコードを設置したものです**
<br>コードから歌詞や色わけを入れ替えるだけで簡単に使えます
## 使い方
<br>*1.まず、レポジトリからindex.htmlと、style.cssをダウンロードしてください
 <br>*   2.次に、そのファイルらをサーバーにアップロードしてください
  <br>*  3.index.htmlの中の歌詞の部分と 歌詞についている class名(例:scoups)などを該当するメンバーに変えてください
   <br>* 4.完成です

 ## 機能
1.ミュージックビデオを見れます
<br>2.メンバーカラー表を画像として保存もできます
<br>3.歌詞の上をマウスを乗せたり指を置くと名前が歌詞の上に表示される
